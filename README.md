# NutriPlanIA

Aplicación móvil Android desarrollada como proyecto de Ingeniería en Informática, orientada a la planificación nutricional personalizada mediante inteligencia artificial.

## Descripción

NutriPlanIA es un sistema inteligente que permite a los usuarios mejorar sus hábitos alimenticios mediante el análisis de datos personales y la generación de recomendaciones nutricionales personalizadas.

## Funcionalidades principales
- Registro del perfil del usuario

- Generación de planes alimenticios personalizados, utilizando algoritmos de recomendación basados en inteligencia artificial para generar planes de alimentación adaptados al perfil del usuario.

- Registro de consumo alimentario

- Análisis de alimentos mediante imágenes, identificando alimentos a partir de fotografías y estima calorías y macronutrientes utilizando modelos de inteligencia artificial.

- Registro de actividad física

- Control de hidratación

- Análisis de hábitos alimenticios

- Generación de recomendaciones inteligentes, entregando sugerencias de mejora continua basadas en el análisis del perfil, hábitos y actividad física.

- Visualización de estadísticas y progreso

- Resumen diario integral

## Tecnologías utilizadas

- Kotlin
- Android Studio
- Firebase (Firestore, Authentication)
- API de Inteligencia Artificial de OpenAI
